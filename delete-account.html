Account Delete Policy

Focus On Career એપ અને વેબસાઇટમાં વપરાશકર્તાને પોતાનું એકાઉન્ટ ડિલીટ કરવાની સંપૂર્ણ સુવિધા આપવામાં આવે છે. વપરાશકર્તાની ગોપનીયતા અને માહિતીની સુરક્ષા અમારી પ્રથમ પ્રાથમિકતા છે.

App માંથી Account Delete કરવાની પ્રક્રિયા

  • વિદ્યાર્થી એપના Profile / Settings વિભાગમાં જઈને Account Delete કરી શકે છે.
  • Account Delete કરતા જ વપરાશકર્તાની તમામ માહિતી સિસ્ટમમાંથી દૂર કરવામાં આવે છે.
  • Delete થયા પછી Course access, Mock Test data અને User profile ફરીથી ઉપલબ્ધ રહેશે નહીં.
  • આ પ્રક્રિયા તરત જ અમલમાં આવે છે.

Website દ્વારા Account Delete Request

જો વપરાશકર્તા એપમાં Login કરી શકતા ન હોય તો Website દ્વારા પણ Account Delete Request કરી શકે છે.

  • Email અથવા WhatsApp દ્વારા Request કરી શકાય છે.
  • Admin દ્વારા ચકાસણી કર્યા બાદ 48 કલાકની અંદર Account Delete કરવામાં આવશે.
  • Delete થયા પછી તમામ વ્યક્તિગત માહિતી સિસ્ટમમાંથી દૂર કરવામાં આવશે.

માહિતી સંબંધિત નોંધ

  • Account Delete થયા બાદ માહિતી પુનઃપ્રાપ્ત કરી શકાશે નહીં.
  • Payment history માત્ર accounting હેતુ માટે record તરીકે રહી શકે છે.
  • User data કોઈ તૃતીય પક્ષ સાથે શેર કરવામાં આવતું નથી.

Account Delete માટે સંપર્ક કરો

કોઈપણ મદદ માટે નીચેના માધ્યમ દ્વારા સંપર્ક કરી શકો છો.
